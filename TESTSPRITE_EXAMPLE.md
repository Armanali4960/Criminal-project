# TestSprite Usage Example

## Overview
This document demonstrates how to use TestSprite tools to generate and execute tests for the Criminal Detection System.

## Step 1: Code Summary Generation
First, we analyze the codebase to understand the technology stack and features:

```bash
# Generate code summary
testsprite_generate_code_summary --projectPath="c:\Users\tusha\Desktop\Arman Mulani - Copy\criminal_detection_system"
```

This creates a `code_summary.json` file with information about:
- Tech stack: Python, Django, OpenCV, SQLite
- Features: Citizen dashboard, image upload, police dashboard, authentication

## Step 2: Standardized PRD Generation
Next, we generate a Product Requirements Document:

```bash
# Generate PRD
testsprite_generate_standardized_prd --projectPath="c:\Users\tusha\Desktop\Arman Mulani - Copy\criminal_detection_system"
```

## Step 3: Test Plan Generation
We generate a test plan based on the PRD:

```bash
# Generate backend test plan
testsprite_generate_backend_test_plan --projectPath="c:\Users\tusha\Desktop\Arman Mulani - Copy\criminal_detection_system"
```

This creates a test plan with test cases like:
- TC001: citizen_dashboard_page_display
- TC002: image_upload_and_processing
- TC003: police_dashboard_page_display

## Step 4: Test Execution
Finally, we execute the tests:

```bash
# Execute tests
testsprite_generate_code_and_execute --projectName="criminal_detection_system" --projectPath="c:\Users\tusha\Desktop\Arman Mulani - Copy\criminal_detection_system" --testIds=[]
```

## Benefits of Using TestSprite
1. **Automated Test Generation**: Creates comprehensive test plans based on your codebase
2. **Structured Testing**: Organizes tests by features and requirements
3. **Easy Execution**: Runs tests and generates detailed reports
4. **Time Saving**: Reduces manual test creation effort
5. **Consistency**: Ensures all features are tested systematically

## Files Generated by TestSprite
- `code_summary.json` - Codebase analysis
- `standard_prd.json` - Product requirements document
- `testsprite_backend_test_plan.json` - Backend test cases
- `testsprite-mcp-test-report.md` - Final test report
